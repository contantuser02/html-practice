
# 🚀 HTML Advanced Elements

Welcome to the **HTML Advanced Elements** section of my HTML Practice Repository.

This folder contains practical examples of advanced HTML tags that help create more semantic, interactive, and user-friendly web pages. Every example is written using clean HTML with proper comments for better understanding and future revision.

---

## 📚 Topics Covered

- 📈 Progress Tag (`<progress>`)
- 📊 Meter Tag (`<meter>`)
- 🖼️ Picture Tag (`<picture>`)
- 🖼️ Figure & Figcaption (`<figure>`, `<figcaption>`)
- 🔍 DataList (`<datalist>`)
- 🧮 Output Tag (`<output>`)
- 📂 OptGroup (`<optgroup>`)
- 🖍️ Mark Tag (`<mark>`)
- 📝 Abbreviation Tag (`<abbr>`)
- 📍 Address Tag (`<address>`)
