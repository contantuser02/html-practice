# 📚 College Library Website

A simple and fully functional **College Library Website** built using **pure HTML5**.

This project was created as part of my **HTML Practice Series** to improve my understanding of HTML structure, semantic tags, forms, tables, navigation, and page organization before learning CSS and JavaScript.

---

## 🌟 Features

- 🏠 Home Section
- 📖 About Library
- 📚 Popular Books Section
- 🏛 Engineering Departments
- 📝 Library Membership Form
- ⏰ Library Timings
- 📜 Library Rules
- 🔍 Search Bar (UI)
- 🗺 Google Maps Integration
- 📞 Contact Information
- 🔗 Navigation using Internal Anchor Links

---

## 🛠 Technologies Used

- HTML5
- Semantic HTML
- Tables
- Forms
- Lists
- Images
- Google Maps Embed
- Anchor Navigation
