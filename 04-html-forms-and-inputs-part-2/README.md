# 📝 HTML Forms and Inputs - Part 2

Welcome to **HTML Forms and Inputs - Part 2**! 🚀

This folder is a continuation of my HTML learning journey. It contains practical examples of various HTML form elements and input-related tags that are commonly used while building interactive web pages.

The purpose of this section is to understand how different input controls work and how they can be used to collect user information effectively.

---

## 📂 Files Included

| File | Description |
|------|-------------|
| 📄 InputTag.html | Introduction to the HTML `<input>` element |
| 🏷️ Label.html | Demonstrates the `<label>` tag and its usage |
| 🔑 Name&Value.html | Understanding the `name` and `value` attributes |
| 🔘 Radioinput.html | Radio button examples |
| 📊 Range.html | HTML range slider input |
| 📋 Select.html | Drop-down selection using the `<select>` tag |
| 📝 TextArea.html | Multi-line text input using `<textarea>` |
| 🌐 iframe.html | Embedding webpages using the `<iframe>` tag |
