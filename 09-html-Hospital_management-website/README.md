# 🏥 Your Hospital Management Website

A professional **Hospital Management Website** built using **pure HTML**. This project is designed to practice HTML by creating a real-world multi-page hospital website featuring healthcare services, doctor information, appointments, departments, patient resources, and contact details.

The website focuses on providing an organized, user-friendly interface while strengthening HTML fundamentals through a complete website project.

> 📚 This project is part of my HTML Learning Journey.
